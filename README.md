# lightbulb

A webapp that controls the lighting within my room.  Runs on a Raspberry Pi 
hooked up to GPIO with the [PowerSwitch Tail II](https://www.sparkfun.com/products/10747).

(In)effectively answers the question, "How many lines of Python does it take 
to turn on a lightbulb?" ;)
